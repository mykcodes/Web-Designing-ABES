Web Designing: Course Curriculum

Course code: 25VA151 | Semester: Odd 2025

Course Overview

This introductory course teaches the fundamentals of building websites. You will learn HTML basics, simple page structure, and how to organize content using lists and semantic elements. The course aims to make you comfortable with creating clear, accessible pages using HTML.

Important notes: attendance is mandatory, assignments are weekly, and projects are team-based.

Learning Outcomes

  1. Understand HTML structure and common tags.
  2. Build semantic pages using headings, paragraphs, and lists.
  3. Create forms and embed multimedia.
  4. Document and present project work clearly.

Weekly Syllabus

  1. Week 1 — Introduction to HTML
    • Structure
    • Headings,paragraphs
    • Practical
  2. Week 2 — Text formatting: Speacial characters
    • Bold, italic, underline
    • HTML entities(&,<,>,..)
  3. Week 3 — Lists
    • Unordered lists
    • Ordered lists for example:
      1. Plan structure
      2. Write markup
      3. Test in browser
  4. Week 4 — Description lists semantic grouping

    We will practice the structure for glossaries and definitions.

  5. Week 5 — Project Review
    • Group project: Build a mini website (3 pages)
    • Final review presentations

Key Terms (Description List)

HTML
HyperText Markup Language — the standard markup language for documents designed to be displayed in a web browser.
Element
A combination of a start tag, content, and an end tag.
Entity
A special sequence that represents a character.

Assignment Example (Mixed & Nested Lists)

Task: Create a 1-page personal profile using only HTML.

  • Requirements:
    1. Include at least three headings (h1, h2, h3)
    2. Include one paragraph with bold, italic and underline
    3. Include one unordered list of hobbies
    4. Include a description list for contact details (email, phone)
  • Optional:
    • Use HTML entities for special characters
    • Comment your code
